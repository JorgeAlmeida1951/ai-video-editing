## 🚀 How to Edit Your First Video

The software functions through a simple four-step process.

### Step 1: Import Your Media
Launch the application. Drag your video file from your folder into the main workspace. The AI will immediately begin analyzing your footage to detect speech and visual cues.

### Step 2: Generate Subtitles
Click the Transcribe button. The system converts your audio into text. You can review the transcript on the side panel. If the AI misses a word, click the text box to edit the content manually. Any changes you make in the text appear as cuts in your source video.

### Step 3: Use Auto-Editing Tools
Use the search bar to find specific words or phrases. Highlight the text and press delete to remove those sections from your video file. The AI automatically adjusts your video timeline to bridge the gaps. This prevents jump cuts and keeps your video flow natural.

### Step 4: Export Your Final Project
Once you finish your edits, click the Export button. Choose your preferred video resolution and file format. Select a location on your hard drive to save the exported file. The software processes your edit and creates a single video file ready for publishing.

## 🛠 Troubleshooting

Common issues often have simple solutions. Check this list if the application behaves unexpectedly.

- If the application stalls during import, ensure you have sufficient hard drive space.
- If the AI fails to transcribe, check your internet connection. The transcription engine requires a stable link to fetch the language data.
- If the video preview stutters, check if other intensive programs run in the background. Use the Task Manager to close browser tabs or other large applications.
- If the installer fails, right-click the file and select Run as Administrator.
- If you see a security warning, click More Info and select Run anyway. This happens because the installer is a standalone executable.

## 📝 Frequently Asked Questions

Do I need a subscription for the AI tools?
The core version includes local processing for basic editing. Some advanced features contact our online servers to ensure high accuracy.

Can I edit files larger than 4K?
Yes, the application supports high-definition video files. Large files take longer to analyze, so please wait for the progress bar to finish before attempting to edit.

Where does the application store my projects?
The software creates a folder titled AI-Video-Projects in your Documents directory. You may change this path in the Settings menu under Preferences.

Does this software save my raw data to the cloud?
No, your raw video files remain on your local hard drive. Only the audio transcript segment gets sent to our system to generate text data. This ensures your sensitive footage stays private.

Can I use this software offline?
Basic trimming works without an internet connection. However, generating AI subtitles and advanced audio cleanup requires access to our processing servers.
